###学习引导
+ 坚定自己的学习目标，短计划与长计划。
+ 培养对课程的兴趣，有兴趣才能逐步深入。
+ 遇到问题别太钻牛角尖，或许就是一些约定俗成的规定。
+ 踏实学习，好好坚持。

> 你学习2个小时的时间不难，难的是你每天坚持学习2个小时，如果你能坚持下去，短则3个月，长则半年，你肯定能学有所成！
